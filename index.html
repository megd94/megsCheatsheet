Meg's CSS styling Cheatsheet

Learn how to use the most important CSS properties

CSS property Values Description
font-family serif, sans serif, Garamond, 'Courier New', Helvetica There is an endless list of fonts. You can use Google fonts to pick the type of font you would like. You can then use a link in the HTML to the fonts or within the CSS. Always give a last option of 'serif' or 'sans serif' as a backup if no other fonts can render!
font-size 10px, 2em, 3rem There are default font sizes for h1, h2, p tags etc. However you can change this by specifying the font size. Using px will specify a specific size. Using em or rem will multiply the default size. E.g. 1.5em would be 1.5 X the normal size for that tag.
font-weight 100 = thin, 200 = extra light, 300 = light, 400 = normal, 500 = medium, 600 = semi-bold, 700 = bold, 800 = extra bold. You can use either the numerical or written value for the weight property. By default the text is set to normal
background color #FFFEEE, ivory, hsl(56, 100%, 97%), RGB(255, 254, 238) This is an example of one colour: ivory. You can either write 'ivory'. You can also use the # hex value. This uses letters to indicate how much of each colour is used. The HSL value uses: hue, saturation, lightness and sets each to a percentage to create the colour. Or you can use RGB which stands for red, green, blue and each colour is set to an amount out of 255. You can also add an 'a' on the end for alpha. This will then set the opacity for this we use between 0-0.1.
Color #FFFEEE, ivory, hsl(56, 100%, 97%), RGB(255, 254, 238) This is an example of one colour: ivory. You can either write 'ivory'. You can also use the # hex value. This uses letters to indicate how much of each colour is used. The HSL value uses: hue, saturation, lightness and sets each to a percentage to create the colour. Or you can use RGB which stands for red, green, blue and each colour is set to an amount out of 255. You can also add an 'a' on the end for alpha. This will then set the opacity for this we use between 0-0.1.
text-align left, right, center, justify, inherit Self explanatory. This will align the text in whatever position we set. Inherit will be whatever the parent's element is. Justify spaces words out so that content starts at left edge and ends at the right edge.
height auto, 10px, 100px, 1.5em If auto the element will automatically adjust its height to allow its content to be displayed correctly. If (r)em then the content will times by however much, so 1.5em is 1.5 times the normal set height.
width auto, 10px, 100px, 1.5em If auto the element will automatically adjust its height to allow its content to be displayed correctly. If (r)em then the content will times by howesver much, so 1.5em is 1.5 times the normal set width.
border 1px, dotted, red, 5px, solid, black Border is shorthand propery for: border-width, border-style (required), border-color. The options for the border style are: none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset, initial, inherit
border-radius 5%, 50%, 2em, 25px This allows us to round the edges of our boxes. Can use (r)em, px or %. If you use 50% it will create a circle
padding 10px, 2em/ padding: 10 30px 40px 20px/td> Padding is part of the box model. This is the space between the content and its border. Padding is shorthand for padding-top, padding-bottom, padding-left, padding-right . If set like padding: 10px 20px; The top and bottom will be 10px. The left and right 20px.
margin 10px, 2em/ margin: 10px 20px 30px 40px Margin is part of the box model. It sets the space between the edge of the page and the border of the box. Margin is shorthand for margin-top, margin-bottom, margin-left, margin-right. We can set margin: 20px 30px; To mean a 20px top and bottom, 30px left and right and so on.
overflow Visible, hidden, scroll, auto The overflow property specifies whether to clip the content or to add scrollbars when the content of an element is too big to fit in the specified area. If visible, the overflow is not clipped. The content renders outside the element's box. If hidden, the overflow is clipped, and the rest of the content will be invisible. If scroll the overflow is clipped, and a scrollbar is added to see the rest of the content. If auto - Similar to scroll, but it adds scrollbars only when necessary
box-sizing border-box This property only has one value. If used padding and border will be included in the element's total width and height. This allows us to clearly set height and width, without havingt to calculat extra for padding or borders. It will take the space from within the content itself.
position static, relative, fixed, absolute, sticky Staticis set as default. Elements are not affected by the top, bottom, left, and right properties. relative;is positioned relative to its normal position. Setting the top, right, bottom, and left properties of a relatively-positioned element will cause it to be adjusted away from its normal position.e.g. position: relative; left; 30pxIt will now be 30px away to the left from its normal position. fixed is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. Positon set by left, right etc properties again. Absolute is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).It is removed from normal flow, and can overlap elements. Sticky is positioned based on the user's scroll position.A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed). Yet again uses left, right etc to set the positon where it wills stick.
display none, inline, inline-block, block, grid, flex inline keeps all content on one line. Any height and width properties will have no effect. inline-block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width valuesblock displays an element as a block element It starts on a new line, and takes up the whole width. grid displays an element as a block-level grid container. flex displays an element as a block-level flex container
float left, right, none, inherit The float property is used for positioning and formatting content e.g. let an image float left to the text in a container.
clear none, left, right, both, initial, inherit The clear property controls the flow next to floated elements. It specifies what should happen with the element that is next to a floating element. both indicates it is pushed beneath both left and right floated elements. initial sets to default.
